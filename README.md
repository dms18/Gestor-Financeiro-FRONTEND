# Gestor Financeiro

Aplicativo móvel para gestão financeira de Microempreendedores Individuais (MEIs), desenvolvido com Ionic Angular e Capacitor.

## Descrição

Este aplicativo permite aos MEIs gerenciar suas finanças de forma simples e eficiente, com funcionalidades para controle de receitas, despesas e relatórios básicos.

## Tecnologias Utilizadas

- **Framework**: Ionic Angular
- **Runtime Nativo**: Capacitor
- **Linguagem**: TypeScript
- **Plataformas**: iOS, Android e Web

## Como Executar

1. Instale as dependências: `npm install`
2. Execute no navegador: `ionic serve`
3. Para dispositivos móveis: `ionic capacitor run android` ou `ionic capacitor run ios`
